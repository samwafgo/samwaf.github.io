IP黑名单

1 IP黑名单

防护墙会阻止在黑名单内的IP的访问

1.1 IP黑名单

支持IP和网段形式 例如：127.0.0.1 , 0.0.0.0/0

1.2 批量添加黑名单IP

见：网站管理->批量任务 批量添加黑名单IP

2 功能简介

IP 黑名单用于设置“拦截名单”：被列入黑名单的 IP 或网段访问受保护网站时，SamWaf 会直接阻止其访问。常用于封禁恶意来源、攻击 IP、异常爬虫网段等。

页面顶部提示：SamWaf 防护墙会阻止在黑名单内的 IP / 网段的访问。

3 操作步骤

3.1 新增黑名单 IP

  1. 点击左上角的 新建黑名单IP 按钮，弹出新增窗口。
  2. 选择 网站（必填），即该黑名单生效的受保护站点。
  3. 填写 Ip（必填），支持单个 IP 或网段（CIDR），例如 1.2.3.4、10.0.0.0/8。
  4. 可在 备注 中填写说明信息（可选）。
  5. 点击 确认 保存。

3.2 编辑 / 删除

  • 在列表每行的 操作 列，点击 编辑 可修改该条黑名单的网站、IP 与备注。
  • 点击 删除 可删除单条黑名单（会弹出确认提示）。

3.3 批量删除与清空

  • 勾选列表前的多选框后，点击 批量删除 可一次性删除所选记录（无选中时按钮不可用）。
  • 点击 清空所有 可清空当前条件下的全部黑名单数据（会弹出二次确认）。

3.4 导出数据

点击 导出数据 按钮可将黑名单导出为 Excel 文件（ipblock.xlsx），便于备份或迁移。

3.5 搜索

页面右上角支持按 网站 和 Ip 进行筛选，输入后点击 查询。

4 字段说明

字段说明
网站黑名单生效的受保护站点，新增/编辑时必填
Ip拦截的 IP 或网段，支持单个 IP 与 CIDR，如 127.0.0.1、0.0.0.0/0，新增/编辑时必填
备注自定义说明信息，可选
创建时间记录的创建时间（列表展示，自动生成）

注意

请谨慎使用 0.0.0.0/0 等大范围网段，避免误封正常访客。如同时配置了 IP 白名单，白名单优先放行。
