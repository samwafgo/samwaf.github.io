# 应用管理

## 功能简介

应用管理用于在 SamWaf 中托管并守护本地业务进程（如 jar 包、可执行程序等）。SamWaf 可以启动、停止、重启被托管应用，按策略在异常退出时自动拉起，并提供日志查看、初始文件上传、升级/回滚以及网络监听信息查看等能力。

页面顶部提示：可信路径（允许托管的应用目录）在 `conf/config.yml` 的 `application_allow_dirs` 字段中配置，修改后需重启 SamWaf 生效。

> 大部分敏感操作（新建、编辑、删除、启动、重启、升级）都需要先输入 **应用操作密码**，详见下文「应用操作密码」一节。

## 操作步骤

### 1. 新建应用

1. 点击左上角 **新建应用**（若尚未输入应用操作密码，会先弹出操作密码确认框）。
2. 在表单中按区块填写配置：

   **基本配置**
   - **应用名称**（必填）
   - **启动命令**（必填，如 `java -jar app.jar` 或 `./myapp`）
   - **工作目录**：留空则默认 `data/applications/{code}/`；表单会显示「实际目录」，若填写绝对路径会提示确认该路径存在且包含可执行文件，相对路径则相对 SamWaf 安装目录。
   - **环境变量**：格式 `KEY=VALUE,KEY2=VALUE2`（逗号分隔）
   - **启用状态**：开 / 关
   - **自动启动**：是 / 否

   **停止配置**
   - **停止方式**：信号（SIGTERM/taskkill）或自定义命令
   - **停止命令**：仅当停止方式为自定义命令时显示
   - **停止等待**：等待停止的秒数（1–300）

   **重启策略**
   - **重启策略**：不自动重启 / 异常退出时重启 / 总是重启
   - **重启等待**：策略非「不自动重启」时显示，单位秒
   - **最大重启次数**：策略非「不自动重启」时显示，0 表示不限

   **日志配置**
   - **内存日志行数**（100–10000）

   - **备注**：可选

   **初始文件（可选，仅新建时）**
   - **选择文件**：可上传应用的初始文件到工作目录
   - **SHA256 校验值**：选择文件后自动计算，也可手工填写

3. 点击 **提交** 完成创建，点击 **取消** 放弃。

<!-- 图：新建应用表单 -->

### 2. 启动 / 停止 / 重启

在列表对应应用行的操作列：

- **启动**：当应用未在运行时显示，发送启动指令。
- **停止**：当应用运行中时显示，发送停止指令。
- **重启**：发送重启指令。

指令发送后会显示「处理中...」，系统会轮询状态直至到达目标状态后刷新列表。

### 3. 查看日志

- 点击 **查看日志** 打开日志窗口，查看应用的内存日志，并可清空日志。

### 4. 升级 / 回滚

- 点击 **升级/回滚** 打开升级窗口，可上传新版本文件并选择操作类型（仅上传 / 上传并升级，升级流程为：停止 → 备份 → 替换 → 重启），也可从备份列表中选择历史版本进行回滚。

<!-- 图：升级/回滚窗口 -->

### 5. 查看网络

- 点击 **网络** 查看该应用的监听端口与连接 IP 等网络信息（最多显示 1000 条连接记录）。

### 6. 变更记录

- 点击 **变更记录** 打开抽屉，查看该应用的变更历史。按钮上的徽章用于提示未读的变更条数。

### 7. 编辑 / 删除

- **编辑**：修改应用配置（同新建表单字段，密码确认通过后打开）。
- **删除**：在确认弹窗中确认后删除该应用。

## 字段说明

| 字段 | 说明 |
|------|------|
| 应用名称 | 应用的显示名称，必填。 |
| 运行状态 | 运行中 / 已停止 / 已崩溃。 |
| PID | 当前进程 ID。 |
| 重启次数 | 已发生的重启次数。 |
| 启用状态 | 开 / 关。 |
| 自动启动 | 是 / 否，SamWaf 启动时是否自动拉起该应用。 |
| 重启策略 | 不自动重启 / 异常退出时重启 / 总是重启。 |
| 启动命令 | 启动应用的命令，必填。 |
| 工作目录 | 应用运行目录，留空默认为 `data/applications/{code}/`。 |
| 环境变量 | 形如 `KEY=VALUE,KEY2=VALUE2` 的环境变量，逗号分隔。 |
| 停止方式 | 信号（SIGTERM/taskkill）或自定义命令。 |
| 停止命令 | 自定义停止方式时使用的命令。 |
| 停止等待 | 等待应用停止的秒数（1–300）。 |
| 重启等待 | 重启前等待的秒数。 |
| 最大重启次数 | 自动重启的上限，0 表示不限。 |
| 内存日志行数 | 内存中保留的日志行数（100–10000）。 |
| SHA256 校验值 | 上传文件的校验值，选择文件后自动计算。 |
| 备注 | 应用说明信息，可选。 |

## 应用操作密码

新建、编辑、删除、启动、重启、升级等操作需要验证 **应用操作密码**。

- 该密码与登录密码不同，首次启动时自动生成并打印在启动日志中，也可在 `conf/config.yml` 的 `application_op_password` 字段中查看。
- 在操作密码确认框中输入密码，可勾选 **本次会话内记住密码**，记住后一段时间内无需重复输入。
- 密码错误会提示「操作密码错误，请重新输入」。

## 常见问题

- **工作目录填写后不生效 / 路径不可用？** 可托管的目录受 `conf/config.yml` 的 `application_allow_dirs`（可信路径）限制，修改该字段后需重启 SamWaf 生效。
- **不知道应用操作密码？** 查看首次启动日志，或在 `conf/config.yml` 的 `application_op_password` 字段中查看。
