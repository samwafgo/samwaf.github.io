# 数据分析（访问分析）

## 功能简介

数据分析页面以**世界地图**和**词云**两种可视化方式，展示选定时间范围内各国家/地区的访问分布情况，帮助你直观了解流量来源以及攻击/放行行为的地理分布。

- 左侧为世界地图：颜色越深表示该国家/地区的访问量越大，地图支持鼠标缩放与拖拽漫游，鼠标悬停可查看具体地区名称和访问数值。
- 右侧为词云：以国家/地区名称的大小反映访问量权重，访问量越大字号越大。

## 操作步骤

1. 在顶部 **日期** 选择要统计的时间范围，可使用预设的「最近 7 天 / 最近 3 天 / 今天」，也可自定义起止日期。
2. 在日期右侧的下拉框选择要统计的**行为类型**（全部 / 阻止 / 放行 / 禁止）。
3. 点击 **查询**，地图与词云会根据所选条件刷新数据。
4. 在地图上移动鼠标可查看各地区的访问明细；地图为空说明所选范围内没有对应数据。

## 字段说明

| 字段 | 说明 |
|------|------|
| 日期 | 统计的时间范围，默认最近 7 天；支持预设区间与自定义区间。 |
| 行为类型 | 按防御结果筛选统计数据，可选「全部 / 阻止 / 放行 / 禁止」。 |
| 世界地图 | 按国家/地区聚合的访问量分布，颜色深浅代表访问量大小，支持缩放、漫游和悬停查看。 |
| 词云 | 以字号大小表示各国家/地区的访问量权重，访问量越大字越大。 |

## 常见问题

- **地图/词云没有数据怎么办？** 请确认所选日期范围内确实存在对应行为类型的访问记录，并适当放宽时间范围或将行为类型切换为「全部」后再查询。
- **行为类型的含义？** 「阻止」「放行」「禁止」对应 WAF 对该次访问的处理结果，选择具体类型可只统计该类访问的地理分布。
