# 系统配置

## 功能简介

系统配置用于以「键值对」的形式维护 SamWaf 的系统级配置项。每个配置项由配置项（键）、配置值（值）和备注组成，可在本页面进行新建、查询、编辑和删除。

> 系统配置为高级功能，配置项会影响系统行为，请在了解含义后再进行修改。

<!-- 图：系统配置列表 -->

## 操作步骤

### 新建配置

1. 点击左上角的 **新建系统配置**。
2. 在弹窗中填写 **配置项**（必填）、**配置值**，并可填写 **备注**。
3. 点击 **确定** 保存。

### 查询配置

1. 在右上角的搜索区域输入 **配置项** 或 **备注** 关键字。
2. 点击 **搜索** 进行过滤。

### 编辑 / 删除配置

- 在列表操作列点击 **编辑**，修改配置项、配置值或备注后点击 **确定** 保存。
- 在列表操作列点击 **删除**，在确认弹窗中确认后删除该配置。

<!-- 图：新建/编辑系统配置弹窗 -->

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 配置项 | 配置的键名，必填，用于唯一标识一项配置。 |
| 配置值 | 配置项对应的值。 |
| 备注 | 对该配置项的补充说明。 |
| 创建时间 | 该配置项的创建时间。 |
| 操作 | 提供「编辑」和「删除」操作。 |
