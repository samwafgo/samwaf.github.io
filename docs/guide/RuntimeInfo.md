# 运行信息

## 功能简介

运行信息（运行参数）页面以「名称 / 值」列表的形式，集中展示 SamWaf 当前的运行状态与系统环境信息，便于排查问题、确认版本、观察队列与 QPS 等运行指标。

页面内容由后端实时生成，进入页面即自动加载，无需手动操作。

<!-- 图：运行信息列表页 -->

## 操作步骤

1. 进入「系统设置 → 运行参数」页面。
2. 页面会自动加载并展示当前运行信息列表。
3. 重新进入或刷新页面即可获取最新数据。

## 字段说明

| 名称 | 说明 |
|------|------|
| 最后处理 log 时间 | 引擎最近一次从队列取出并处理日志的时间，可用于判断日志处理是否积压。 |
| Goroutine 数量 | 当前进程内的 Goroutine（协程）数量，反映并发负载。 |
| 系统类型 | 运行所在操作系统类型（如 windows / linux）。 |
| 系统架构 | 运行所在 CPU 架构（如 amd64 / arm64）。 |
| 编译器版本 | 构建该程序所用的 Go 版本。 |
| Win7 内核 | 是否运行在 Win7 兼容内核模式（是 / 否）。 |
| 系统已运行时长 | SamWaf 自启动以来的运行时长（天 / 时 / 分 / 秒）。 |
| 软件版本 | 当前 SamWaf 的版本名称。 |
| 软件版本 Code | 当前版本对应的内部版本号。 |
| 当前 QPS | 当前实时请求每秒处理量。 |
| 当前队列数 | 各内部队列的待处理数量（主数据 / 日志数据 / 统计数据 / 消息队列）。 |
| 当前日志队列处理 QPS | 日志队列的实时处理速率。 |
| 当前 web 端口使用列表 | 当前正在使用的业务 Web 端口列表。 |
| 当前隧道端口使用列表 | 当前正在使用的隧道端口列表。 |

> 说明：以上指标均为实时只读信息，仅用于查看与诊断，本页面不提供修改操作。

## 常见问题

- **数据多久更新一次？** 每次进入或刷新页面时由后端实时生成，展示的是当前时刻的状态。
- **队列数一直很大正常吗？** 队列数代表待处理任务量，瞬时偏高属正常；若长期居高不下，可能是磁盘 / 数据库写入存在瓶颈，建议结合「最后处理 log 时间」一起判断。
- **在哪里修改数据库 / 缓存等运行配置？** 运行配置（如数据库 driver、缓存类型）需编辑 `conf/config.yml` 并重启 SamWaf 生效，本页面仅用于查看运行状态。
