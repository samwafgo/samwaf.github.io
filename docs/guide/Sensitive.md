# 敏感词管理

## 功能简介

敏感词检测用于对请求或返回内容中的特定敏感词进行检测与处理。SamWaf 防护墙会阻止包含敏感词的内容，或对返回内容中的敏感词进行替换，从而实现内容合规、防止敏感信息外泄。

每条敏感词可单独设置检测方向（请求 / 返回 / 所有）和执行动作（阻止 / 替换）。

![敏感词](/images/Sensitive.png)

## 操作步骤

1. 进入「敏感词」页面。
2. 点击左上角的 **新建敏感词** 按钮，打开新增弹窗。
3. 在弹窗中填写以下内容：
   - 填写 **内容**（要检测的敏感词）。
   - 选择 **检测方向**（请求 / 返回 / 所有）。
   - 选择 **执行动作**（阻止 / 替换）。
   - 填写 **备注**（可选）。
4. 点击 **确认** 保存。

> 列表上方可通过「内容」进行筛选搜索；勾选多条记录后可使用 **批量删除**，或使用 **清空所有** 一次性删除全部敏感词数据。

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 内容 | 要检测的敏感词文本。 |
| 检测方向 | 敏感词的检测范围，可选：请求、返回、所有。 |
| 执行动作 | 命中敏感词后的处理方式，可选：阻止、替换。 |
| 备注 | 自定义说明（可选）。 |
| 创建时间 | 记录的创建时间（系统自动生成）。 |

### 检测方向说明

- **请求**：检测客户端发送给服务器的请求内容。
- **返回**：检测服务器返回给客户端的响应内容。
- **所有**：同时检测请求和返回内容。

### 执行动作说明

- **阻止**：命中敏感词时阻止该次访问。
- **替换**：命中敏感词时对内容中的敏感词进行替换处理（通常用于返回内容的脱敏）。

## 常见问题

- **替换动作适合什么场景？** 当不希望直接阻断访问，只想隐藏返回内容中的敏感词时，选择「替换」更合适；而需要拦截含敏感词的请求时，选择「阻止」。
- **敏感词是否区分网站？** 敏感词为全局列表，不按单个网站区分，请通过检测方向和执行动作控制其作用范围。
