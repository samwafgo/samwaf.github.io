# 账号管理

## 功能简介

账号管理用于维护登录 SamWaf 管理端的后台账号。在「账号管理 → 账号列表」中可以查看所有账号，并完成新建账号、编辑账号、重置密码、重置双因素认证（2FA）、删除账号等操作。每个账号可分配不同的角色，以区分管理权限。

<!-- 图：账号列表页面 -->

## 操作步骤

### 1. 查询账号

- 在页面右上角的 **登录帐号** 输入框中填入关键字，点击 **查询** 即可按登录帐号筛选列表。
- 列表展示登录帐号、角色、备注、添加时间等信息。

### 2. 新建账号

1. 点击页面左上角的 **新建账号** 按钮，打开新建弹窗。
2. 填写下列字段：
   - **登录帐号**（必填）
   - **角色**：从下拉中选择（超级管理员 / 系统管理员 / 安全管理员 / 审计管理员）
   - **登录密码**（必填）
   - **状态**：数值，默认 1
   - **备注**：可选说明
3. 点击 **确定** 完成创建；点击 **关闭** 取消。

<!-- 图：新建账号弹窗 -->

### 3. 编辑账号

1. 在列表对应账号行点击 **编辑**。
2. 可修改 **登录帐号**、**角色**、**状态**、**备注**。
3. 点击 **确定** 保存。

> 编辑弹窗中不包含密码修改，修改密码请使用「重置密码」功能。

### 4. 重置密码

1. 在列表对应账号行点击 **重置密码**。
2. 弹窗中 **登录帐号** 会自动带出，依次填写：
   - **超级管理员密码**：用于校验当前操作者身份。
   - **新密码**
   - **确认密码**：需与新密码一致，否则会提示「两次输入的密码不相同，请检查」。
3. 点击 **确定** 完成密码重置。

<!-- 图：重置密码弹窗 -->

### 5. 重置 2FA

- 在列表对应账号行点击 **重置2FA**，即可清除该账号已绑定的双因素认证信息。账号下次登录时将仅需密码（如需再次启用 2FA，可在双因素认证页面重新绑定）。

### 6. 删除账号

1. 在列表对应账号行点击 **删除**。
2. 在确认弹窗中点击确认完成删除。

> 默认管理账号不允许删除。

## 字段说明

| 字段 | 说明 |
|------|------|
| 登录帐号 | 用于登录管理端的账号名，必填。 |
| 角色 | 账号角色，可选：超级管理员、系统管理员、安全管理员、审计管理员。 |
| 登录密码 | 新建账号时设置的登录密码，必填。 |
| 状态 | 账号状态值，新建时默认 1。 |
| 备注 | 账号说明信息，可选。 |
| 添加时间 | 账号创建时间，系统自动记录。 |
| 超级管理员密码 | 重置密码时用于身份校验。 |
| 新密码 / 确认密码 | 重置密码时填写，两者必须一致。 |

## 常见问题

- **两次输入的密码不相同？** 重置密码时，**新密码** 与 **确认密码** 必须完全一致。
- **无法删除某个账号？** 默认管理账号受保护，不允许删除。
- **遗失认证器无法登录？** 可通过「重置2FA」清除对应账号的 2FA 绑定，恢复仅密码登录。
