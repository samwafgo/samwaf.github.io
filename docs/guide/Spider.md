# 蜘蛛（爬虫识别分析）

## 功能简介

蜘蛛分析页面用于统计选定时间范围内各类**爬虫（蜘蛛）/访客**的访问占比，帮助你识别站点上搜索引擎爬虫、机器人与正常访客的构成情况。

页面以**饼图**直观展示各爬虫类型的访问占比，右侧配合**明细表格**列出每种类型的访问次数与占比，并可分页查看。

<!-- 图：蜘蛛分析页面，左侧饼图、右侧明细表格 -->

## 操作步骤

1. 在顶部选择要统计的**日期范围**（支持预设的「最近 7 天 / 最近 3 天 / 今天」或自定义起止日期，默认统计当天）。
2. 在网站下拉框选择要查看的**网站**；可选择「全局网站」查看所有站点合计，下拉框可清空。
3. 点击 **查询**，饼图与明细表格会根据所选条件刷新。
4. 在右侧表格中按页查看各爬虫类型的具体访问次数与占比。

## 字段说明

| 字段 | 说明 |
|------|------|
| 日期范围 | 统计的时间范围，默认当天；支持预设区间与自定义区间。 |
| 网站 | 选择要统计的站点，可选「全局网站」查看全部站点合计。 |
| 爬虫类型 | 识别出的爬虫/访客类别名称（如各搜索引擎爬虫、正常访客等）。 |
| 访问次数 | 该类型在所选范围内的访问次数。 |
| 占比 | 该类型访问次数占所选范围总访问量的百分比。 |

## 常见问题

- **没有数据怎么办？** 请确认所选网站在该日期范围内有访问记录，并适当放宽时间范围或切换到「全局网站」后重新查询。
- **占比是怎么计算的？** 占比 = 该爬虫类型访问次数 ÷ 所选范围内总访问次数 × 100%。
