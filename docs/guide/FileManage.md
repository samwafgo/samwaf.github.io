# 文件管理

## 功能简介

文件管理页面用于展示 SamWaf 系统运行过程中产生的文件信息（如日志、数据库、证书等），并支持删除其中允许删除的文件，方便管理磁盘空间。系统会对文件进行保护标记：标记为「不可删除」的文件无法在本页面删除，以避免误删关键数据。

<!-- 图：文件管理列表 -->

## 操作步骤

1. 打开 **文件管理** 页面，系统会自动加载文件列表。
2. 点击右上角 **刷新** 可强制重新扫描并获取最新的文件信息；点击 **重置** 可清除当前的过滤与排序条件。
3. 通过点击表头进行排序（默认按字节大小倒序），或点击表头的过滤图标按 **文件名**、**文件路径**、**描述**、**可删除** 进行过滤。
4. 对于状态为「可删除」的文件，点击操作列的 **删除**，在确认弹窗中确认后即可删除；状态为「不可删除」的文件，删除按钮为灰色不可点击。

> 文件删除后无法恢复，请谨慎操作。

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 操作 | 对可删除文件提供「删除」操作，不可删除的文件该操作为禁用状态。 |
| 文件名 | 文件或目录的名称，目录与文件分别以不同图标显示。 |
| 文件路径 | 文件的完整路径。 |
| 文件大小 | 文件大小的可读显示（目录显示为 `-`）。 |
| 字节大小 | 文件大小的原始字节数，用于精确排序。 |
| 描述 | 文件的用途说明。 |
| 可删除 | 文件是否允许删除，分为「可删除」和「不可删除」两种状态。 |
| 修改时间 | 文件的最后修改时间。 |

## 常见问题

- **为什么有的文件不能删除？** 标记为「不可删除」的文件通常为系统运行所必需的关键文件，本页面禁止删除以保证系统正常运行。
- **删除后能恢复吗？** 不能。删除操作不可逆，请确认后再删除。
