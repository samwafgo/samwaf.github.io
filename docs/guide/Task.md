# 任务管理

## 功能简介

任务管理用于查看与管理系统内置的计划任务（如 SSL 证书自动续期、清理过期防火墙 IP 封禁规则、清理历史数据等）。每个任务可设置执行间隔与具体时刻，支持手工执行并查看任务运行日志。

<!-- 图：任务管理列表 -->

## 操作步骤

### 1. 查看任务列表

进入页面后即可看到内置任务列表，包含任务名、间隔单位、值、具体时刻、内部标识、星期等信息。

### 2. 编辑任务

在任务对应行点击"编辑"，可调整执行周期等参数：

- "间隔单位"选择每秒 / 每分钟 / 每小时 / 每天 / 每周。
- "值"为间隔数量（如间隔单位为每小时、值为 2，表示每 2 小时执行一次）。
- "具体时刻"可指定执行时间，例如 `14:30`，或用分号分隔多个时间 `08:00;20:00`。
- 当"间隔单位"为"每周"时，会出现"星期"选项，用于指定星期几执行。

> 注意：编辑时"内部标识"不可修改（系统内置任务标识）。

填写完成后点击"确认"保存。

### 3. 手工执行

在任务对应行点击"手工执行"，可立即运行该任务一次，无需等待计划时间。

### 4. 查看 / 清空日志

点击"查看日志"打开日志对话框：

- 可查看该任务对应的日志文件内容及路径。
- 点击"刷新"手动刷新；打开"自动刷新"开关后每 5 秒自动刷新一次。
- 点击"清空日志"可清空当前任务的日志。

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 任务名 | 任务的名称 |
| 间隔单位 | 执行间隔单位：每秒 / 每分钟 / 每小时 / 每天 / 每周 |
| 值 | 间隔数量，与间隔单位组合表示执行频率 |
| 具体时刻 | 指定执行时间，如 `14:30`；多个时间用分号分隔，如 `08:00;20:00` |
| 内部标识 | 系统内置任务的标识（编辑时不可修改），对应具体的内置任务，如 SSL 证书自动续期、清理过期防火墙 IP 封禁规则等 |
| 星期 | 仅当间隔单位为"每周"时生效，指定星期日至星期六中的某天执行 |
