# 系统日志

## 功能简介

系统日志用于记录管理端的操作行为，便于审计与回溯。每条日志记录了操作类型、操作内容和发生时间。本页面以列表形式展示系统日志，并支持查看每条日志的详细内容。

<!-- 图：系统日志列表 -->

## 操作步骤

1. 打开 **系统日志** 页面，列表会自动加载日志记录。
2. 在右上角的搜索框中输入关键字，可对系统日志进行检索。
3. 点击列表操作列的 **详情**，在弹窗中查看该条日志的完整 **操作类型** 与 **操作内容**。
4. 查看完毕后点击 **关闭** 退出详情弹窗。

<!-- 图：系统日志详情弹窗 -->

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 操作类型 | 该条日志对应的操作类别。 |
| 操作内容 | 操作的具体内容描述。 |
| 创建时间 | 该条日志的记录时间。 |
| 操作 | 提供「详情」入口，查看该条日志的完整信息。 |
