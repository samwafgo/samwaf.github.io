# SamWaf网站防火墙
SamWaf是一款适用于小公司、工作室和个人网站的免费轻量级网站防火墙，一键启动，支持Linux，Windows 64位

![SamWaf网站防火墙概览](/images/overview.png)

## 主要功能：
- 完全独立引擎，防护功能不依赖IIS,Nginx
- 自定义防护规则，支持脚本和界面编辑
- 支持白名单访问
- 支持IP黑名单
- 支持URL白名单
- 支持限制URL访问
- 支持指定界面数据隐私输出
- 支持CC频率访问
- 支持全局一键配置
- 支持分网站单独防护策略

## 已测试支持的平台
- Centos 64位
- Windows 2008r2 64位
- Windows10 64位
