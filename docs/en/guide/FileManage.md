# File Management

## Overview

The File Management page lists files generated by SamWaf during operation (such as logs, databases, and certificates) and lets you delete those that are allowed to be removed, making it easier to manage disk space. Files are protected with a deletable flag: files marked **Non-deletable** cannot be removed on this page, preventing accidental loss of critical data.

<!-- Image: File management list -->

## Steps

1. Open the **File Management** page; the file list loads automatically.
2. Click **Refresh** at the top-right to force a rescan and fetch the latest file information; click **Reset** to clear the current filters and sorting.
3. Sort by clicking a column header (sorted by size in bytes, descending, by default), or use the filter icon in the headers to filter by **File Name**, **File Path**, **Description**, or **Can Delete**.
4. For files with the **Deletable** status, click **Delete** in the action column and confirm in the dialog. For **Non-deletable** files, the delete button is greyed out and disabled.

> Deleted files cannot be recovered. Proceed with caution.

## Field Reference

| Field | Description |
| --- | --- |
| Operation | Provides a **Delete** action for deletable files; disabled for non-deletable files. |
| File Name | The name of the file or directory; directories and files are shown with different icons. |
| File Path | The full path of the file. |
| File Size | Human-readable file size (directories show `-`). |
| Size Bytes | The raw size in bytes, used for precise sorting. |
| Description | A note on what the file is used for. |
| Can Delete | Whether the file may be deleted: either **Deletable** or **Non-deletable**. |
| Modified Time | The last modification time of the file. |

## FAQ

- **Why can't some files be deleted?** Files marked **Non-deletable** are usually critical files required for the system to run, so deletion is blocked on this page to keep the system working.
- **Can deletion be undone?** No. Deletion is irreversible, so confirm before deleting.
