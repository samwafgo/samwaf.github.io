# System Configuration

## Overview

System Configuration maintains SamWaf's system-level settings as key-value pairs. Each entry consists of a Configuration Item (key), a Configuration Value (value), and Remarks. You can add, search, edit, and delete entries on this page.

> System Configuration is an advanced feature. These items affect system behavior, so only change them once you understand what they mean.

<!-- Image: System configuration list -->

## Steps

### Add a configuration

1. Click **Add New System Configuration** at the top-left.
2. In the dialog, fill in the **Configuration Item** (required) and the **Configuration Value**, and optionally **Remarks**.
3. Click **Confirm** to save.

### Search configurations

1. Enter a **Configuration Item** or **Remarks** keyword in the search area at the top-right.
2. Click **Search** to filter.

### Edit / delete a configuration

- Click **Edit** in the action column, change the item, value, or remarks, then click **Confirm** to save.
- Click **Delete** in the action column, then confirm in the dialog to remove the configuration.

<!-- Image: Add / edit system configuration dialog -->

## Field Reference

| Field | Description |
| --- | --- |
| Configuration Item | The key of the configuration; required, uniquely identifies an entry. |
| Configuration Value | The value of the configuration item. |
| Remarks | Additional notes about the entry. |
| Create Time | The creation time of the entry. |
| Operation | Provides **Edit** and **Delete** actions. |
