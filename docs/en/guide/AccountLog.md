# Account Log

## Overview

The Account Log records the operations performed by admin accounts, making auditing and tracing easier. From **Account Management → Account Log** you can view each operation record, including the operation account, operation type, operation content, and creation time, and inspect the details of a single record.

<!-- Image: Account log list -->

## Steps

### 1. Search Logs

- Enter a keyword in the search box at the top of the page to filter records.
- The list shows the operation account, operation type, operation content, and creation time, with pagination support.

### 2. View Details

- Click **Details** on the target record row. The dialog displays the record's **Operation Account**, **Operation Type**, and **Operation Content** in read-only form.
- Click **Close** to exit the dialog when finished.

<!-- Image: Account log detail dialog -->

## Field Reference

| Field | Description |
|-------|-------------|
| Operation Account | The login account that performed the operation. |
| Operation Type | The category of the operation. |
| Operation Content | The detailed description of the operation. |
| Operation Time | The time the log entry was generated. |
