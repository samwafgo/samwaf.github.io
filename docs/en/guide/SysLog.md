# System Log

## Overview

The System Log records management-side operations for auditing and traceability. Each entry records the operation type, operation content, and the time it occurred. This page lists the system logs and lets you view the full details of each entry.

<!-- Image: System log list -->

## Steps

1. Open the **System Log** page; the log entries load automatically.
2. Enter a keyword in the search box at the top-right to search the system logs.
3. Click **Details** in the action column to view the full **Operation Type** and **Operation Content** of that entry in a dialog.
4. Click **Close** to exit the details dialog when done.

<!-- Image: System log details dialog -->

## Field Reference

| Field | Description |
| --- | --- |
| Operation Type | The category of the operation recorded by this entry. |
| Operation Content | A description of what the operation did. |
| Create Time | The time the entry was recorded. |
| Operation | Provides a **Details** entry to view the full information of the log entry. |
