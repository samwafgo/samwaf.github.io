              常见于当SamWaf和web应用服务器在同一台服务器下。

              # 2 登录相关

              # 2.1 密码忘记了

              SamWaf64.exe resetpwd

              # 2.2 口令失效

              为了安全自v1.3.10-beta.4版本，将token增加实效性。

              可以依据实际情况更改：

              菜单路径：系统设置->参数设置

token_expire_time
管理平台令牌有效期，单位分钟（默认5分钟）

              令牌有两种失效情况：第一种是令牌到期失效， 第二种是帐号使用不同IP登录

              # 2.3 2FA安全码无法登录

              SamWaf64.exe resetotp
